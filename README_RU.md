# SiteMoney Audit

Готовый микро-продукт для быстрого заработка на аудитах сайтов малого бизнеса.

Что делает:
- скачивает сайт и проверяет HTML-сигналы: CTA, контакты, формы, отзывы, локальные сигналы, schema, analytics, mobile viewport, HTTPS, скорость ответа;
- считает `Conversion score` и примерную утечку денег;
- строит продаваемые приоритеты правок;
- генерирует email, follow-up, DM и Telegram-сообщение;
- предлагает 3 пакета услуги;
- ведет локальную историю, CRM-следующее действие и экспорт HTML/JSON/CSV.

Важно: оценка денег нужна для продажи разговора. Она не гарантирует доход.

## Локальный запуск

На этом компьютере можно просто запустить:

```powershell
.\RUN_LOCAL.ps1
```

Открыть: `http://127.0.0.1:8787`

Если Node.js установлен обычным способом:

```powershell
npm install
npm run build
npm start
```

Открыть: `http://127.0.0.1:8787`

Проверка версии: `http://127.0.0.1:8787/__version`

## Быстрый способ продавать

1. Собрать 20 сайтов в одной нише и городе.
2. Вставить URL в очередь.
3. Запустить очередь.
4. Открыть самые слабые отчеты.
5. Отправить готовое сообщение владельцу.
6. Продавать сначала `Quick Fix`, не большой редизайн.

## Деплой

Проект готов к Railway:

```powershell
npm install
npm run build
npm start
```

Railway использует `railway.json`, `Procfile`, `/health` и `/__version`.
Production entrypoint: `node app-server.mjs`.

Если Railway показывает `ERROR packages field missing or empty`, проверь что в корне есть
`pnpm-workspace.yaml` с блоком:

```yaml
packages:
  - "."
```
